These vocabulary questionnaires contain only the words used in this experiment + a list of words planned for a future English version (in preparation).
All words are in French, and filled in by a French-speaking parent (or other French-speaking caregiver in case of two English-speaking parents).

== Columns description ==
subject_id: 	ID of child subject, same as used for exp. data
word_original: 	target word (correct spelling)
stage: 			which stage of the experiment it appears in, options: training, test or unused (i.e. words for the English exp.)
word: 			target word, with assimilation applied only in test items - those meant for the English exp. are instead marked as [eng]
knows:			0 = doesn't know, 1 = knows but doesn't use, 2 = knows and uses
